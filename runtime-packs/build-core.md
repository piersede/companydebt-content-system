# Build Core

Compact execution pack for page builds and build-adjacent script work.

## Priorities

- preserve build correctness
- preserve quality gates
- keep editorial context inspectable
- do not weaken checks to make publishing easier

## Rules

- use the system-decided runtime context for the page and task
- expose routing when useful for debugging
- preserve verification dates, scope notes, and caution notes where required
- build success does not override editorial failure
