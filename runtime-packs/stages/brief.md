# Stage Pack: Brief

Goal:

- define who the page is for, what real question it answers, what decision it supports, and what a satisfying outcome looks like

Rules:

- be operational, not vague
- avoid generic persona language
- name the real tension, not just the topic
- declare the primary entity and intent so the page does not compete with a sibling owner
- identify the main audience lane explicitly
- declare the page type explicitly and start from the canonical outline for that type
- write the brief so the rest of the workflow can stay anchored to it
