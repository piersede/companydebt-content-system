# Runtime Pack Benchmarks

This directory measures runtime-pack savings against representative editorial workflows.

Quality supremacy rule:

- token savings are subordinate to editorial quality
- if a runtime-pack change lowers trust, sharpness, decision usefulness, or human-authorship credibility, reject it
- if a runtime-pack change weakens entity ownership, freshness handling, or decision-critical call-out-box discipline, reject it
- if a runtime-pack change weakens page-type structure discipline or decision-layer endings, reject it

Non-compressible human-authorship checklist:

- concrete scenes remain explicit
- earned `you` and `we` remain explicit
- lived friction remains explicit
- evaluative bite remains explicit
- rhythm and sentence texture remain explicit
- asymmetrical editorial lines remain explicit
- mild UK texture remains available where natural
- moral clarity remains available where warranted

Benchmark rule:

- do not count a token reduction as a success unless the runtime layer still preserves the full human-authorship model during execution

Run:

- `python scripts/benchmark_runtime_packs.py`
